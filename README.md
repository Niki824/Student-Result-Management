# Student Result Management System in C

## Features
- Store student marks using arrays
- Calculate total marks
- Calculate percentage
- Display grade
- Uses functions for modularity
- Saves results to a text file

## Technologies
- C Programming
- Arrays
- Functions
- File Handling

## Bonus Feature
- File handling is used to save student results into student_results.txt
